# Session Log — Append-Only

Every meaningful AI-assisted work session is logged here. Append at the end of every session. Do not edit or delete prior entries; if a fact in an earlier entry was wrong, add a correcting entry below it.

Format per session:

```
## YYYY-MM-DD

- What was done (one line per action)
- Files created or modified (with paths)
- Decisions made or deferred
- Recommended next session actions
```

The AI reads this at the start of every session to understand current case state. Keep entries factual and dated. This file plus `00-MASTER-INDEX.md` plus `09-chronology/` are the three places the AI orients itself from.
