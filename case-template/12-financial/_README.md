# 12-financial

Case expenses (legal, travel, missed work, services compliance costs) and damages tracking for any future federal civil-rights action.

Keep receipts. Track time. Mileage logs. Lost wages with employer documentation. Out-of-pocket costs for court-ordered services (UAs, evaluations, classes, supervised visits).

Naming: `YYYY-MM-DD_Category_Vendor_Description.ext`.

When building a §1983 damages claim, the bill of particulars assembles from this folder. Date-of-injury and category granularity matters — don't lump.
