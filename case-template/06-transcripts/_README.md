# 06-transcripts

Hearing transcripts (official court reporter transcripts) and recording transcripts (transcribed from your one-party-consent recordings — see `recording-protocols` skill before you record anything).

Naming: `YYYY-MM-DD_Type_Source.md` (e.g. `2026-03-14_Hearing_TPR-status.md`, `2026-04-02_Recording_caseworker-home-visit.md`).

If a transcript is generated from an audio file, the audio original belongs in `../05-evidence/` (hashed). The transcript here references the original by its hash.

For analysis of party-opponent admissions or contradictions in a transcript, use `/extract-admissions`.
