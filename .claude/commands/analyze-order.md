# Analyze Court Order

Apply Prompt 11 from the CPS Parent's AI Prompt Library to the order at: $ARGUMENTS (or, if no argument, ask the user to paste the order).

Produce:

1. ORDER TYPE
2. FINDINGS (every finding of fact, quoted)
3. CONCLUSIONS (every conclusion of law, quoted)
4. REQUIREMENTS ON ME — what I must do, by when, how compliance is measured
5. REQUIREMENTS ON THE AGENCY — often overlooked; the agency has obligations too
6. RIGHTS PRESERVED
7. DEADLINES with calendar dates and consequences
8. NEXT HEARING (date, type, what will be decided)
9. APPEAL RIGHTS (with deadline)
10. AMBIGUITIES (unclear or contradictory language)
11. ENFORCEMENT GAPS
12. ABSENT ELEMENTS that a complete order should contain in this jurisdiction (statutory required findings, reasonable efforts findings)

Save the analysis to `07-drafts/analysis/order-<basename>-<YYYYMMDD>.md`.

Calendar the deadlines in `09-chronology/deadlines.md`.

Log in `_session-log.md`.
